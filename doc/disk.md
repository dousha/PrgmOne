# 磁盘结构定义（草案）

## 游戏内存储格式

游戏内仅允许玩家直接对数据区进行读写，程序区则只能通过烧录与拷贝工具读写。

数据区的起始地址在游戏内映射为 0x0.

## 实际存储格式

每份磁盘拥有一个独立的位于插件数据目录中的文件。

```
+ plugins
|_ PrgmOne
   |_ disks
      |_ <creator>-<uuid>.bin
	  |_ ...
```

文件格式如下：

```
+------------------+ 0x0
|文件限制大小 (KiB)| 4
+------------------+ 0x4
|文件数据区开始位置| 4
+------------------+ 0x8
|      .text       | ???
+------------------+ ???
|      .data       |
+------------------+ 
```
