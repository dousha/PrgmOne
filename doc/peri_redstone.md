# 外设系统 - 红石

## 概述

此外设系统负责 PrgmOne 与 Minecraft 红石系统的兼容。

## 组件

| 组件名		| 允许的位置			|
| ------------- | --------------------- |
| 红石中继器	| 与主机齐平的 8 个位置	|

## 协议

本外设使用通用的外设通信方式。

### 设置红石信号

此操作会打开中继器的红石信号输出。

```
--> rson
<-- 1
```

此操作会关闭中继器的红石信号输出。

```
--> rsoff
<-- 0
```

## 限制

* 本外设的最大操作频率为 5Hz.
