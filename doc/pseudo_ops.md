# 伪助记符定义（草案）

| 伪助记符				| 意义								| 注释								|
| --------------------- | --------------------------------- | --------------------------------- |
| .string addr, str		| 于内存 addr 处初始化字符串 str	| 字符串自动以 \0 结尾				|
| .skip addr, n, fill	| 于内存 addr 处填充 n 个 fill		|									|
| .import name			| 导入 name 代码					| 代码书必须能在玩家物品栏中找到	|
